# Blog

This repository contains the source code for a blog project focused on enhancing my skills in web development. The application includes:

- A frontend (views, styles)
- Backend logic (JavaScript)
- Routing and templates
- Package configuration for dependencies

## Features

- Create a blog
- Edit a blog
- Delete a blog
- Read a blog

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Bootstrap-Icons
- method-override
- EJS
- Express.js
- Node.js
